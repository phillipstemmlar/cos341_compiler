==================================JAR FILE==========================================
To compile the code into a JAR file, use

	make jar

The JAR file will be available in the bin/ directory.

To run the JAR file, called COS341_u18171185_P01.jar, use

	java -jar COS341_u18171185_P01.jar <inputfile>

==================================JAVA FILE=========================================

To compile the code into Java Class files, use

	make

The Java Class files will be available in this directory.

The Main Java Class is the Compiler Class. To run it, use

	java Compiler <inputfile>
